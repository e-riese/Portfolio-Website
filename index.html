Determining the Required Vaccination Coverage in Israel for the Pfizer-BioNTech Vaccine to Achieve Herd Immunity from COVID-19

the Pfizer-BioNTech vaccine was modelled as a commodity experiencing diminishing returns and It was analyzed with multiple linear regression models using Stata. This resulted in the required value of 70% vaccinated population for herd immunity.
